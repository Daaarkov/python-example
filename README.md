# python-example

A simple example in python that create how much file do you want 

***

# Usage

1. Install [python](https://www.python.org/downloads/)
1. Clone repository 
    ```bash
    git clone https://github.com/Daaarkov/python-example.git
    ```
1. Change directory
    ```bash
    cd python-example
    ```
1. Run the file
    ```bash
    python main.py
    ```
    or 
    ```bash
    python3 main.py
    ```
